During my second semester, I developed a "Java-based Snake and Ladders game" as a project, incorporating essential software development concepts. The game featured a "login and signup system" to allow users to create and manage their accounts securely. Players could roll the dice virtually, navigate the board based on game rules, and experience automated movement of tokens according to "snakes and ladders positions". This project enhanced my understanding of "object-oriented programming, user authentication, and interactive game mechanics" in Java.

I initially created a "College of Science and Technology website" prototype using "Figma", focusing on UI/UX design principles to ensure a visually appealing and user-friendly interface. After completing the design phase, I advanced the project by developing the website using "HTML, CSS, and JavaScript", bringing the prototype to life with responsive layouts and interactive elements. This process allowed me to gain hands-on experience in both "web design and front-end development", enhancing my skills in "prototyping, coding, and user experience optimization".

I have been working on a project called Quill, utilizing HTML and CSS, guided by the YouTube channel Future Full Stack. My learning is based on their video titled "The Most Detailed HTML Full Course on YouTube! HTML Tutorial for Beginners + Notes + Summary Cards." This project has allowed me to strengthen my front-end development skills by implementing structured HTML layouts and styling them with CSS for a responsive and visually appealing design.
